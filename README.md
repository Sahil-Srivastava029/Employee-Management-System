# Employee Management System

A responsive Employee Management System built using **HTML**, **CSS**, and **JavaScript**. This project allows users to register, manage, search, and delete employee records through an interactive interface. Employee data is stored using **Local Storage**, so records remain available even after refreshing the page.

## рџ“‹ How to Use

1. Fill in the Employee Registration Form.
2. Click **Register Employee**.
3. The employee will appear in the Employee Records table.
4. Use the search box to search employees by name.
5. Use the department dropdown to filter employees.
6. Click the **Delete** button to remove an employee.
7. Dashboard statistics update automatically.
8. Employee records are automatically saved using Local Storage.
